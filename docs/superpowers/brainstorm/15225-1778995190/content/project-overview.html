📋 项目核心信息

基于 Expo Go 开发的 TOTP/HOTP 身份验证器 App，采用 Claymorphism（黏土形态）设计风格。

🎨 Claymorphism 设计核心特征

黏土卡片

双重阴影

外阴影 + 内阴影模拟物理黏土质感

马卡龙色系

粉/蓝/紫/绿/橙柔和配色

弹簧动效

弹簧动效

cubic-bezier(0.34, 1.56, 0.64, 1)

📱 主要页面结构

  • 🏠 账号列表主页（含 FAB、空状态）
  • ➕ 添加账号动作菜单（扫码/手动/剪贴板）
  • 📝 手动输入表单页
  • ⚙️ 设置页（分组列表）
  • 📷 扫码页面
