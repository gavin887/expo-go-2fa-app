📁 项目结构

目录结构
📁 src/
├── features/
├── accounts/ # 账号管理（增删改查、列表展示）
├── otp/ # OTP 算法（TOTP/HOTP 生成、时间校准）
├── security/ # 安全（加密存储、应用锁、生物识别）
├── import-export/ # 导入导出（备份、恢复、分享）
└── settings/ # 设置（主题、语言、高级选项）
├── shared/
├── components/ # 通用 UI 组件（ClayCard、TimerRing、Toggle 等）
├── hooks/ # 自定义 Hooks
├── utils/ # 工具函数（Base32 编解码、剪贴板等）
└── theme/ # 主题配置（颜色、阴影、动画）
├── navigation/ # 路由配置
├── context/ # 全局 Context Providers
└── App.js # 应用入口

🔑 核心数据流

账号数据 → 加密存储 → Context 状态 → UI 展示 → 倒计时自动刷新

数据流向
1. 用户输入
扫码/手动/剪贴板
2. 解析 & 验证
otpauth URL → Secret
3. 加密存储
expo-secure-store
4. 状态更新
Context dispatch
5. UI 渲染
验证码 + 倒计时
